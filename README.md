## 📝 项目介绍

4o-image-web-deploy 是一个基于 OAI 生图 API 开发的专业图片创作平台，可用于商业化网站运营。系统兼容 OAPI 生图格式的中转服务和直连服务。

## 🛠️ 技术栈

### 后端
- Java SpringBoot 3
- MySQL
- Redis

### 前端
- Vue 3
- Element Plus
- 绘图页面采用纯原生 HTML

## 📦 快速安装

```bash
curl -sSfL https://raw.githubusercontent.com/xiaomifengD/4o-image-web-deploy/refs/heads/main/quick-install.sh | bash
```
